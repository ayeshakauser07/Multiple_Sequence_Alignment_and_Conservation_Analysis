# Multiple_Sequence_Alignment_and_Conservation_Analysis

**Overview**

This script analyzes a multiple sequence alignment (MSA) file generated by Clustal Omega, computing per-position conservation scores, pairwise percent identity between sequences, and gap statistics — useful for comparing how conserved or divergent a gene/protein is across species.

**How to Use It**

Get your sequences (protein or nucleotide) for the gene/species you want to compare — e.g. from NCBI.
Go to Clustal Omega (EBI), paste/upload your sequences, and run the alignment.
Download the result in "Clustal with numbers" (.aln-clustal_num) format.
Place the downloaded file in the same folder as this script, and update the alignment_file variable to match its filename.
Run the script top to bottom.

**What This Pipeline Does**

Loads the Clustal alignment file and reports the number of sequences and alignment length.
Computes a per-column conservation score and plots a conservation profile across the alignment.
Builds a pairwise percent identity matrix and visualizes it as a heatmap.
Counts gaps per sequence to identify poorly aligned or highly divergent species.
Auto-detects a reference sequence (e.g. human) and reports the most similar and most divergent species relative to it.
Saves summary metrics and the identity matrix to text/CSV files.

**Why This Is Helpful**

Quantifies conservation instead of just eyeballing an alignment — useful for spotting functionally important (highly conserved) regions.
Works for any gene or protein family — just supply a different alignment file.
Useful for comparative genomics and evolutionary biology coursework/research.
Outputs (heatmap, conservation plot, CSV) are ready to drop into a report or presentation.

**Notes**

All sequences in the alignment should be the same type (all protein or all nucleotide).
Make sure the alignment_file variable matches your downloaded filename exactly.
